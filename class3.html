Inline vs Block

So far, we have mostly seen "block" elements.

They appear on the next line, like paragraphs.



There are also "inline" elements.

They appear on the same line that they are written on.

Block and Inline Elements

  • CSS divides HTML into two types: inline and block.
  • After block elements, browsers render a new line.
  • Inline elements: img, a, br, em, strong
  • Block elements: p, h1, ul, li, almost everything else

<p>Paragraph with inline <em>emphasized text</em>.</p>

Element: Div

  • A division, or section of content within an HTML page.
  • Block level element. Each div is rendered on a new line.
  • Used to group elements to format them with CSS.
  • Apply IDs and classes to divs to control their styles with CSS.
<div>
   <p>Content<p>
   <p>Content<p>
</div>
<div id="header">
   <h1>Main Heading<h1>
</div>
<div class="sub-content">
   <p>Some more content<p>
</div>

Grouping elements with div

The div tag is used everywhere to group elements together into sections.

Element: Span

  • Inline element.
  • Used to apply styles to text inline (e.g. a sentence inside a paragraph).


The code:

.warning{
  color:red;
}
<p>An inline <span class="warning">warning</span>.</p>

The output:

An inline warning.

Let's Develop It

  • Separate the content on your site into different sections using divs with classes or IDs of "header", "content", and "footer".
  • Style a sentence or a few words within a paragraph using span and a class.

Pseudo-classes, more CSS for links


Pseudo-classes can be used to add hover effects to your links.


CSS pseudo-classes are used to add special effects to some selectors.

Syntax: 
selector:pseudo-class {
  property:value;	
}

Example:

a:hover {
  background-color: blue;
}
Some link

Pseudo-classes, more CSS for links

a:link { /* unvisited link */
  color:#FF0000;
}

a:visited { /* visited link */
  color:#00FF00;
}

a:hover	{ /* mouse over link */
  color:#FF00FF;
}

a:active { /* selected link */
  color:#0000FF;
}

Note: a:hover MUST come after a:link and a:visited in the CSS definition, and a:active MUST come after a:hover!

Let's Develop It

Add pseudo classes to your links.

Box Model

Padding

Space between the border and the content

Adds to the total width of the box.

Padding: Example

Space between the border and the content.

The code:

#content {
  border: 3px solid #ffffff;
  padding: 50px;
}
<div id="content">
  <p>Paragraph with some text.</p>
</div>

The output:

Paragraph with some text.

Padding Property

50 pixels on all sides:

padding: 50px;

10 pixels on top only:

padding-top: 10px;

10 on top, 5 on right, 3 on bottom, 5 on left:

padding: 10px 5px 3px 5px;

Padding

Four values: top right bottom left (TRBL)

padding: 10px 5px 3px 5px;

Three values: top right/left bottom

padding: 30px 20px 10px;

Two values: top/bottom right/left

padding: 15px 10px;

One value: all sides

padding: 50px;

Padding

padding: 10px 20px 30px 40px;

Border

The edge around the box.

Specified as "thickness, style, color."

Border

A solid red border:

border: 1px solid #ff0000;

A thick, dotted, black top border:

border-top: 4px dotted #000000;

Two different border styles together:

border-top: 4px dotted #000000;
border-bottom: 1px solid #ff0000;

Border - Other Properties

You can specify each border property separately, or all three together.

border-width: 10px;
border-style: dashed;
border-color: #666666;

Possible border styles:

  • solid
  • double
  • dotted
  • dashed
  • groove
  • ridge
  • inset
  • outset

Margin

The whitespace around the box that separates it from other elements.

Margin: Example

The code:

#content {
  border: 3px solid #ffffff;
  margin-top: 30px;
  padding: 10px;
}
<h1>Heading</h1>
<div id="content">  
  <p>Paragraph with some text.</p>
</div>

The output:

Heading

Paragraph with some text.

Margin

15 pixels on all sides:

margin: 15px;

10 on top, 5 on right, 3 on bottom, 5 on left:

margin: 10px 5px 3px 5px;

10 pixels on top:

margin-top: 10px;

Auto Margin

If a margin is set to auto on a box that has width, it will take up as much space as possible.

Centered:

margin: auto;
width: 300px;

Flush-right:

margin-left: auto;
margin-right: 5px;
width: 300px;

Property: Width

Sets the width of an element.

#content {
  width: 800px;
  padding: 20px;
  border: 1px solid #000000;
}

Remember: padding and borders add to the total width!

Total width of the #content element: 800px + 40px + 2px

Property: Height

Sets the height of an element.

#content {
  height: 800px;
  width: 800px;
  padding: 20px;
  border: 1px solid #000000;
}

Note: Elements generally expand to fit the height of your content automatically, so used less often.

Box Model: Using the Inspector

Tables

Tables are a way to represent complex information in a grid format.

Tables are made up of rows and columns.

<table>
  <tr> 
    <th>Head</th>
    <th>Head</th>
  </tr>
  <tr> 
    <td>Data</td>
    <td>Data</td>
  </tr>
</table>

Head Head
Data Data

Tables: Examples

Tables can be styled with CSS to add zebra striping or to highlight important rows/columns.
